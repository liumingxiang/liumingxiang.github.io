本文首先说明使用精简的CSS命名规则的意义，为各位提供学习的动机。

最后，提供一些基础的CSS命名优化的案例，从实例去具体认识其奥妙之处。

精简的命名规则

从我对命名规范的理解，可以演绎出命名规范的意义所在：

  • 富有含义的命名规定对于开发者来说，更容易读懂
  • 精简的命名规范能够提高代码的整洁度和用户体验
  • 精简的含义不仅仅包括命名的简洁性，还包括浏览器对页面的渲染速度，即良好的命名规范可以提高页面渲染速度
  • 良好的命名规则会在精确性和复用性上权衡
